# Project Memory

This folder is the repo-side memory scaffold for cross-agent and
cross-conversation continuity.

Subfolders:

- `status/`: where the latest phase status summaries should live
- `decisions/`: why important project decisions were taken
- `handoffs/`: short handoff notes for future agents
- `inventories/`: inventories of datasets, reports, and external dependencies

The plugin-based shared memory layer will be configured separately, but this
repo structure gives all agents a stable on-disk anchor point.
