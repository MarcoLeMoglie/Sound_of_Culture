# Phase 3 Directive: Validation Against External Culture Measures

## Goal

Validate the music-based culture measure against external culture measures
available across US states.

## Main tasks

- identify external state-level culture measures
- harmonize geography and timing
- build validation-ready datasets
- estimate descriptive and formal validation exercises

## Required outputs

- cleaned validation datasets
- code used to harmonize and test validation
- a plain-English report in Overleaf:
  - `phase_03_validation/`

## Mandatory report content

- external data description and provenance
- matching and harmonization logic
- validation strategy
- robustness checks
- kept vs discarded specifications
- code snippets for main construction and estimation steps
