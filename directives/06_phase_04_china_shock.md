# Phase 4.2 Directive: China Shock

## Goal

Study whether exposure to the China shock affects the culture measure derived
from music.

## Main tasks

- build or ingest China-shock exposure data
- harmonize exposure geography and timing
- estimate the causal design consistent with the economics literature being
  followed

## Required outputs

- cleaned shock data
- merged analytical dataset
- estimation scripts
- a plain-English report in Overleaf:
  - `phase_04_causal_shocks/02_china_shock/`

## Mandatory report content

- source provenance
- mapping from raw shock data to the analytical sample
- identification logic
- assumptions and limitations
- kept vs discarded designs
- Stata or Python code snippets for the final retained design
